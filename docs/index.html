Welcome To Fibonacci Test Project!

Requirements

  • Git download
  • Docker download

Run project in docker

git clone https://github.com/vasyl-chyypesh/fibonacci-test.git
cd fibonacci-test
docker compose up --build

Access the API documentation and test interface

Once the Docker containers are running, open your browser and navigate to http://localhost:8080 to view the interactive API documentation and try out the available endpoints via the UI.

Test via curl

POST data with a number:

curl -d '{"number":7}' -H "Content-Type: application/json" -X POST http://localhost:3000/input

This will send a response with a ticket ID in the body:

{ "ticket": 1 }

GET data by ticket ID:

curl http://localhost:3000/output/1

This will send a response with result data in the body:

{ "ticket": 1, "inputNumber": 7, "fibonacci": "13" }

Health check

curl http://localhost:3000/health

Returns 200 with a status payload:

{ "message": "OK", "time": "2026-06-03T00:00:00.000Z" }
